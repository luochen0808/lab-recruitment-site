本团队长期专注于纳米探针与神经信号检测领域，拥有十余年纳米探针设计、分子界面修饰、信号放大及生物芯片集成化的深厚积累，相关成果已成功转化并服务于哈佛、耶鲁、斯坦福等顶尖机构。现面向脑机接口前沿方向，重点开展多模态纳米探针无损脑机接口研究；基于世界领先的纳米生物感受/调节器技术，构建集脑部电信号、多模态生化指标实时监测与颅内药物靶向疗控于一体的诊疗一体化脑机接口平台。核心技术突破在于：研发微纳低损探针，实现电、光、化、力学多模态复合传感与闭环刺激，解决现有植入器件普遍存在的组织损伤、信号漂移及长期稳定性不足等临床瓶颈。该平台有望为脊髓损伤、脑卒中后遗症等千万级患者群体提供精准、安全的神经功能修复方案，具有重大科研价值与产业化前景。目前团队在单细胞无损电生理记录、活体神经精准调控、单分子相互作用分析等方面已具备成熟理论与技术储备；本方向将培养与研究内容深度交叉融合脑机接口技术、数字孪生架构、大模型智能赋能、多尺度建模仿真、神经信号处理、纳米生物探针、多模态信息融合等前沿研究板块，学科交叉性强、科研平台完善、课题资源充足。
本次拟招生方向有如下：
招生方向一（生物脑科学与微纳器件方向）：重点聚焦于生物脑科学、脑神经机制与微纳器件研发的交叉融合，有脑科学、生物医药、神经工程等研究背景，或从事微纳器件、纳米生物探针相关研究的考生，核心围绕纳米探针与神经信号检测的生物层面、器件研发层面开展研究，助力诊疗一体化脑机接口平台的生物相容性与器件性能优化。
招生方向二（数字智能与信号处理方向）：侧重数字技术与脑机接口的深度结合，具备脑机接口、数字孪生、大模型智能赋能、神经工程、信号处理相关研究背景的考生，核心围绕脑机接口信号分析、数字孪生架构构建、大模型赋能神经调控、多模态信息融合等方向开展研究，推动脑机接口平台的智能化、精准化升级。
